Quickdraw vs

A project a friend and I started. Both on github, and hosted online at quickdraw-vs.com. It's a game where you compete to see who's drawing can get classified by a NN first. It was my first time training any NN, so I learned a lot. Admittedly, the code was pretty bad though. I used pytorch for the model and we went with fastAPI as a backend since it was pretty easy to pick up. I mostly focused on the model, frontend drawing, and collaborated on system design, but I think both of us touched all areas of the project. We used redis and websockets, and it sort of inspired me to make my own project using them later down the line in Go.
